# Phase 1: Foundation

**Outcome:** "I can read AI-generated code and think like an agent architect"

**Estimated time:** ~7–8 hours
**Dependencies:** None — this is the starting point.

1. **[Lesson-01-Reading-AI-Generated-Python.md](Lesson-01-Reading-AI-Generated-Python.md)** — PRIMM-AI+, the Ten Axioms, dev environment, and how to read AI-generated Python before writing any of it yourself.
2. **[Lesson-02-TDG-Cycle-and-Type-Vocabulary.md](Lesson-02-TDG-Cycle-and-Type-Vocabulary.md)** — Primitive types and collections, plus your first full Test-Driven Generation (TDG) cycle.
3. **[Lesson-03-Agent-Architecture-Fundamentals.md](Lesson-03-Agent-Architecture-Fundamentals.md)** — What separates an agent from a chatbot: the 3+1 architecture, the operational loop, and multi-agent coordination patterns.
4. **[Lesson-04-Agent-Ops-Security-and-Choosing-a-Framework.md](Lesson-04-Agent-Ops-Security-and-Choosing-a-Framework.md)** — Operating and monitoring a live agent, interop/security across systems, and how to choose an SDK.
